    SEPARACIONES (LÍNEAS)


    Para separar un área o un bloque con una línea, se utiliza la etiqueta < hr >, esta atraviesa de extremo a extremo; con la etiqueta < br > realizamos saltos de línea, es como dar un enter.

    Tambien se puede cambiar el ANCHOR, GROSOR, ALINEACIÓN Y COLOR de la línea de la etiqueta < hr > como veremos en seguida.


    línea con etiqueta < hr >


    línea con etiqueta < hr width = #px >
    width nos da la anchura de la línea, en mi caso la puse a 30 px, pero también se puede expresar en porcentajes


    línea con etiqueta < hr size = #>
    size nos define el grosor que tendrá nuestra línea, en este caso le dimos 10 de tamaño.


    línea con etiqueta < hr color = color a nuestra elección>
    color nos define el color que tendrá nuestra línea, en este caso le dimos color oro.


    línea con etiqueta < hr color = color a nuestra elección>
    le dimos color rosa.


    línea con etiqueta < hr align = center width = 50%>
    align al igual que en párrafos nos define la alineación que tendrá nuestra línea, en este caso la alineamos al centro.


    línea con etiqueta < hr align = right >
    en este caso la alineamos la derecha.


    línea con etiqueta < hr align = left >
    en este caso la alineamos la izquierda.


    línea con etiqueta < hr align = left width = 50% color = blue size =25 >
    en este caso la alineamos la izquierda con una anchura del 50%, color azúl y un grozor de 25.



    TABLAS


    Ya conocemos las tablas, así como en word o en excel, se compone de filas, columnas y celdas que podemos armar con las etiquetas:

  • < table >... < /table >
    Esta etiqueta se utiliza para abrir y cerrar la tabla, es necesario y muy importante NO olvidarnos de esta, puesto que si ponemos las etiquetas < td > y < tr > sin < table > será como si NO hubieramos puesto nada.
  • < td >... < /td >
    Esta nos indica las filas ----> que van de manera horizontal.
  • < tr >... < /tr >
    Esta nos indica columnas que van de manera vertical.

    Esta es una tabla centrada de 2 columnas por 3 filas, dentro de cada celda de la columna 1 escribiremos "columna 1- fila X" contando en que X = al número de la fila correspondiente, y dentro de la columna 2 escribiremos el las etiquetas correspondientes.

    columna 1 - fila 1 columna 2 - fila 1
    columna 1 - fila 2 columna 2 - fila 2
    columna 1 - fila 3 columna 2 - fila 3

    Todo este código va dentro de las etiquetas < table border = 2>... < /table>, le dimos el grosor del borde #2 quedando de la siguiente manera:

    < center>< table border="2">
    < tr> < td>columna 1 - fila 1< /td>
    < td>columna 2 - fila 1< /td>
    < /tr>

    < tr>
    < td>columna 1 - fila 2< /td>
    < td>columna 2 - fila 2 < /td>
    < /tr>

    < tr>
    < td>columna 1 - fila 3< /td>
    < td>columna 2 - fila 3 < /td>
    < /tr>
    < center>


    INSERCIÓN DE IMÁGENES


    Así como el texto, las líneas, las tablas y el resto de contenido que se puede introducir en una página, las imágenes se pueden alinear, cambiar el borde de la imagen y cambiar su tamaño. Para insertar una imágen utilizaremos la etiqueta:

    < img src = "dirección donde/se encuentra la imagen">< /img>

    Para cambiar el tamaño usaremos Width (que determina el ancho de la imagen, su valor por default es de 100 puntos de imagen) y height (esta determina la altura de la imagen).


  • Esta tiene los puntos en 50
    código:< img src="imagenes/fuga.png" width=50 height=50>< /img>



  • Esta tiene los puntos en 100
    código:< img src="imagenes/fuga.png" width=100 height=100>< /img>



  • Esta tiene los puntos en 300
    código:< img src="imagenes/fuga.png" width=300 height=300>< /img>


    •

    para cambiar la alineación usaremos "align = Posición" donde posición puede ser: middle (la imagen y el texto quedan alineadas al medio), bottom (la imagen queda a la par del texto, es decir sobre el mismo párrafo), top (El texto queda alineado a la parte superior a la imagen), left (la imagen se desplaza a la izquierda), right(la imagen se desplaza a la derecha).

  • aquí estamos usando "middle" quedando de la siguiente forma
    < img src="pagina prueba/fuga.png"width=100 height=100 align=middle>< /img>


  • aquí estamos usando "bottom" quedando de la siguiente forma
    < img src="pagina prueba/fuga.png"width=100 height=100 align=bottom>< /img>


  • aquí estamos usando "top" quedando de la siguiente forma
    < img src="pagina prueba/fuga.png"width=100 height=100 align=top>< /img>


  • aquí estamos usando "left" quedando de la siguiente forma
    < img src="pagina prueba/fuga.png"width=100 height=100 align=left>< /img>





  • aquí estamos sando "right quedando de la siguiente forma
    < img src="pagina prueba/fuga.png"width=100 height=100 align=right>< /img>





    LISTAS (ORDENADAS, NO ORDENADAS Y DE DEFINICIÓN)


    En el lenguaje de HTML es posible mostrar texto en forma de listas o numeración. Existen 3 tipos de listas que podemos utilizar.

  • NO ORDENADAS: Esta lista muestra un símbolo para indicar los diferentes elementos de la lista.
    Símbolo Código
    • cuadrado
    		< ul type = square>
    < li> Texto a listar
    < li> Texto a listar
    < /ul>
    • disco
    		< ul type = disc>
    < li> Texto a listar
    < li> Texto a listar
    < /ul>
    • círculo
    		< ul type = circle>
    < li> Texto a listar
    < li> Texto a listar
    < /ul>


  • ORDENADAS: Esta lista utilizará un número o letra para indicar cada inciso de la lista.
    Orden Código
    1. numeración (números arábigos)
    		< ol type = 1>
    < li> Texto a listar
    < li> Texto a listar
    < /ol>
    1. listado de letras minúsculas
    		< ol type = a>
    < li> Texto a listar
    < li> Texto a listar
    < /ol>
    1. listado de letras mayúsculas
    		< ol type = A>
    < li> Texto a listar
    < li> Texto a listar
    < /ol>
    1. listado de números romanos
      en minúsculas
    		< ol type = i>
    < li> Texto a listar
    < li> Texto a listar
    < /ol>
    1. listado de números romanos
      en mayúsculas
    		< ol type = I>
    < li> Texto a listar
    < li> Texto a listar
    < /ol>


  • DE DEFINICIÓN: Este tipo de lista incluye dos elementos: por un lado la definición que aparece al borde del lado izquierdo y por el otro el término que aparece tabulado a la derecha.
    Vista Código
    Término a definir
    definición del término
    		< dl>
    < dt>Término a definir
    < dd>definición del término
    < /dl>



    • INSERCIÓN DE VIDEOS


    Para añadir un video o película a una página web a nuestra página web, los formatos más comunes son: ".avi", ".mpg", ".wmv"

    Usaremos la etiqueta < img dynsrc=dirección /del video > < /img> esta etiqueta tiene varios atributos que nos ayudan a tener contro de nuestro video.

    loop=número
    Estas son las veces que deseamos que el video se repita.
    loop=infinite
    Este atributo permite que el video se repita ininterrumpidamente, es decir, cuando acaba vuelve a empezar.
    height y width
    Estos atributos nos definen las dimensiones que tendrá la pantalla de nuestro video.
    start=fileopen
    El video inicia en cuanto carga nuestra página.
    start=mouseover
    El video inicia en cuanto situamos el indicador del mouse sobre el cuadro de video.

¿Qué es ATOM?

Atom es un editor de texto que da vida a las líneas de código de una aplicación o una página web, es soportado por MAC, Linux y Windows y que fué desarrollado por GITHUB.

Este editor de texto es de código abierto, es decir, podemos modificarlo a como nos convenga.

Atom incluye las funciones básicas para integrarse con GIT y cuenta con la capacidad de instalar las diferentes paqueterías que se encuentran en la página oficial.

https://atom.io/packages
